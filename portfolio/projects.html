Face Mask detector:

Introduction

We Introduce a face mask detection model that is based on Artificial Intelligence. The proposed model can be integrated with surveillance cameras to impede the COVID-19 transmission by allowing the detection of people who are wearing masks not wearing face masks. The model is integration between deep learning and classical machine learning techniques with opencv, tensor flow and keras.

Aim of the Project:

*Year 2020 will always be remembered because of a major pandemic which the world is still grappling with, that is Covid 19 Studies have shown that the spread of COVID has been slower at locations where people are wearing mask. In order to study the spread of the virus, determining if masks are properly being worn by people or not, or whether they are being worn at all plays a key role. Our project is based on this line of thought : When presented with an image or a live video, the python based code will label each face in one of the three categories : With Mask, Without Mask, Improperly worn mask

Domain of the project:

The domain of the project is Machine Artificial Intelligence. AI uses various algorithms based on the requirement of the project.

Objective:

To Build a Real Time Application useful to Control the Spread of Virus Building a Real Time Application that can Detect even Facial Occlusions. Classification of Real Time Face Images as Masked and Unmasked Increasing the Number of Images in Dataset in order to Improve the Accuracy of the System Optimization of the Algorithm to Capture and Detect Unmasked Face in Real Time
